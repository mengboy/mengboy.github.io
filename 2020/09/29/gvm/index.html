造轮子——go 版本控制(gvm)

发表于 2020-09-29 更新于 2020-10-02 分类于 go Disqus：

功能列表

  • 支持linux、darwin
  • 获取远程go版本列表 gvm remote
  • 安装指定版本go gvm install [version]
  • 列出本地已下载go gvm list
  • 切换go版本 gvm use [version]
  • 清理本地指定版本go gvm clear [version]
