常见prompt框架

deepseek出现后，AI的势头更加一路不可阻挡， cursor最近又更新了Claude 3.7，编程体验进一步提高，身为码农对未来还有几年职业生涯充满了焦虑。熟话说打不过就加入，AI的浪潮已经阻挡不了，最好的方式就是学习接受，细节算法实现暂且不提门槛太高，还是先从使用上入手，下面整理常见的prompt框架。

常用提示语结构

清华大学DeepSeek PDF中提到的RTGO和CO-START


其他提示语结构

框架名 首字母含义 示例
APE - Action (行动)
- Purpose (目的)
- Execution (执行)		 - 行动: 为新推出的智能手表撰写产品描述
- 目的: 在电商平台上吸引潜在买家并提升产品销量
- 执行: 生成三段不同风格的产品描述，分别突出手表的健康监测、便捷支付和时尚外观功能
ERA - Expectation (期望)
- Role (角色)
- Action (行动)		 - 期望: 生成一篇能够提高社交媒体互动的产品宣传文案
- 角色: 担任社交媒体营销专家
- 行动: 撰写一篇突出产品优势并包含行动号召的简短文案，附带产品链接和互动问题
RTF - Role (角色)
- Task (任务)
- Format (格式)		 - 角色: 资深行业分析师
- 任务: 解析新能源汽车市场趋势
- 格式: 包含数据可视化图表与SWOT分析矩阵的PPT大纲
TAG - Task (任务)
- Action (行动)
- Goal (目标)		 - 任务: 提升长尾词排名
- 行动: 优化TDK标签
- 目标: 3周内自然流量增长20%
CIDI - Context (上下文)
- Instructions (指令)
- Details (细节)
- Input (输入)		 - 上下文: 你是一位金融分析师，需要对某科技公司的季度财报进行分析
- 指令: 分析公司的收入增长、利润率、现金流和主要业务部门表现
- 细节: 报告需包含数据可视化，长度限制在3页内，重点关注同比变化
- 输入: 提供最近4个季度的财务报表数据、行业平均水平和主要竞争对手数据
CARE - Context (上下文)
- Action (行动)
- Result (结果)
- Example (示例)		 - 上下文: 一家新开的咖啡店需要提升社交媒体知名度
- 行动: 策划为期一个月的社交媒体营销活动，包括每日图文更新、互动活动和影响者合作
- 结果: 粉丝数增长50%，每日互动率提升30%，带来线下客流增长20%
- 示例: 参考星巴克的”#PSL季节”社媒活动，通过话题引导和用户原创内容带动参与度
RASE - Role (角色)
- Action (行动)
- Result (结果)
- Example (示例)		 - 角色: 担任一位经验丰富的旅行顾问
- 行动: 为一位计划前往日本旅行两周的客户制定行程计划
- 结果: 生成一份包含每日行程、交通方式、住宿建议和餐饮推荐的详细旅行计划
- 示例: 客户偏好文化体验和自然风光，预算中等
COAST - Context (上下文)
- Objective (目标)
- Actions (行动)
- Scenario (场景)
- Task (任务)		 - 背景: 一家电商公司过去一年客户留存率持续下降
- 目标: 在下个季度将客户留存率提升10%
- 行动: 实施客户忠诚度计划，改进客户服务流程，进行客户沟通个性化
- 场景: 客户反馈表明服务响应速度慢，缺乏个性化关怀，且现有会员体系缺乏吸引力
- 任务: 分析客户流失原因，设计新的客户忠诚度计划，优化客服培训内容，并制定个性化沟通方案
ROSES - Role (角色)
- Objective (目标)
- Scenario (场景)
- Expected solution (期望解决方案)		 - 角色: 担任专业的职业规划师
- 目标: 为一位希望转行到数据分析领域的客户提供职业发展建议
- 场景: 客户目前从事市场营销工作五年，具备一定的商业分析能力，但缺乏数据分析技能和经验
- 期望解决方案: 一份详细的职业发展规划，包括技能提升路径、认证建议和实践项目推荐
RISEN - Role (角色)
- Instructions (指令)
- Steps (步骤)
- End Goal/Expectations (最终目标/期望)
- Narrowing/Novelty (约束/新颖性)		 - 角色: 担任资深营销活动策划专家
- 指令: 为一款新推出的智能家居产品策划一场线上发布会
- 步骤: 详细列出发布会策划步骤，包括活动主题、内容环节、嘉宾邀请、宣传推广、互动环节设计等
- 最终目标/期望: 发布会目标是吸引10,000名观众在线观看，并在活动后一周内提升产品官网访问量50%
- 约束/新颖性: 发布会时长控制在60分钟内，采用虚拟现实技术增加互动性，并结合时下流行的直播带货模式
TRACE - Task (任务)
- Request (请求)
- Action (行动)
- Context (上下文)
- Example (示例)		 - 任务: 为一篇科技博客文章撰写引人入胜的开头段落
- 请求: 开头段落需要吸引读者继续阅读，并概括文章的主题
- 行动: 使用悬念、提问或引人注目的事实来开头
- 上下文: 文章主题是关于人工智能在医疗健康领域的应用，目标读者是对科技和健康领域感兴趣的普通大众
- 示例: 提供几个优秀博客文章开头段落的例子，例如《纽约时报》或《连线》杂志的科技文章
CREATE - Character (角色)
- Request (请求)
- Examples (示例)
- Action (行动)
- Task (任务)
- Execution (执行)		 - 角色: 担任一位资深的社交媒体内容编辑
- 请求: 为一家环保组织撰写一篇关于塑料污染危害的社交媒体帖子
- 示例: 分享类似环保主题的成功社媒案例
- 行动: 结合数据和故事性叙述
- 任务: 在有限字数内传达关键信息
- 执行: 创作引人共鸣的内容并配以适当的多媒体元素
COSTAR - Context (上下文)
- Objective (目标)
- Style (风格)
- Tone (语调)
- Audience (受众)
- Response (回复)		 - 上下文: 一家时尚品牌正在进行新品发布预热
- 目标: 在社交媒体上发布一段文字，预告即将发布的新系列，并引发用户期待
- 风格: 时尚、简约、高级
- 语调: 自信、优雅、略带神秘感
- 受众: 关注时尚潮流的年轻消费者和品牌忠实用户
- 回复: 撰写一段社交媒体文案，字数控制在50字以内

框架使用建议

以上介绍的这些prompt框架各有特点，可以根据不同场景选择合适的框架：

  1. 内容创作类：CREATE、TRACE、COSTAR框架更适合用于文案、文章等创意内容的生成，它们注重风格、语调和受众定位。

  2. 项目规划类：COAST、RISEN框架适合用于项目策划和执行，它们包含了详细的步骤和目标设定。

  3. 分析报告类：CIDI、RTF框架适合用于撰写分析报告和专业文档，它们强调结构化输出和专业性。

  4. 营销策划类：CARE、ERA框架适合用于营销活动策划，它们关注结果导向和实际效果。

  5. 简单任务类：APE、TAG框架结构简单，适合日常小型任务的prompt编写。

在实际使用中，可以根据任务的复杂程度和需求特点，选择合适的框架，甚至将多个框架的优势结合起来，构建更适合自己需求的prompt模板。记住，这些框架仅是参考，最重要的是能够清晰地表达你的需求和期望。
