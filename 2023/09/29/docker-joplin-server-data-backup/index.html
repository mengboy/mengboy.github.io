dockerйғЁзҪІjoplin-serverж•°жҚ®еӨҮд»Ҫй—®йўҳ

еңЁд№ӢеүҚзҡ„дёҖзҜҮж–Үз« дёӯи®°еҪ•дәҶдҪҝз”ЁdockerйғЁзҪІjoplin-serverзҡ„жөҒзЁӢгҖӮжңүжҸҗеҲ°joplin-serverеҸҜд»ҘдҪҝз”ЁSQLiteжҲ–иҖ…PostgresSQLпјҢиҝҷдёӨз§ҚеӯҳеӮЁж–№ејҸеҲҶеҲ«иҜҘеҰӮдҪ•еӨҮд»Ҫж•°жҚ®е‘ўпјҹ

SQLite еӨҮд»Ҫ

й»ҳи®ӨдҪҝз”ЁSQLiteзҡ„ж—¶еҖҷпјҢжңҚеҠЎдјҡеңЁ/home/joplin/packages/serverдёӢеҲӣе»әdb-prod.sqliteж–Үд»¶жқҘеӯҳеӮЁз¬”и®°ж•°жҚ®гҖӮ

жҳҫз„¶пјҢиҝҷз§ҚеңЁdockerйҮҢеҲӣе»әзҡ„ж–Үд»¶пјҢdockerй•ңеғҸдёҖж—Ұй”ҖжҜҒж–Үд»¶е•Ҙзҡ„е°ұйғҪжІЎдәҶпјҢдёҖиҲ¬жҲ‘д»¬йңҖиҰҒжҢҒд№…еҢ–зҡ„ж•°жҚ®йғҪдјҡйҖүжӢ©жҢӮиҪҪеҚ·зҡ„ж–№ејҸжҠҠдё»жңәзӣ®еҪ•жҢӮиҪҪеҲ°dockerй•ңеғҸйҮҢпјҢиҝҷж ·dockerеә”з”Ёе®һйҷ…ж“ҚдҪңе°ұжҳҜдё»жңәзӣ®еҪ•дёӢж–Үд»¶пјҢеҚідҪҝdockerй•ңеғҸй”ҖжҜҒдё»жңәзӣ®еҪ•дёӢзҡ„ж–Үд»¶д№ҹдёҚдјҡеҸ—еҪұе“ҚпјҢйӮЈжҺҘдёӢжқҘе°қиҜ•дҪҝз”ЁжҢӮиҪҪи·Ҝеҫ„еӯҳеӮЁз¬”и®°ж•°жҚ®гҖӮ
йҰ–е…ҲпјҢйҰ–ж¬ЎйғЁзҪІзҡ„ж—¶еҖҷжҢӮиҪҪеҘҪзӣ®еҪ•

жңҚеҠЎеҗҜеҠЁеҗҺиҝӣе…Ҙе®№еҷЁеҶ…е‘Ҫд»ӨиЎҢпјҢжҠҠеҲқе§ӢеҢ–зҡ„db-prod.sqliteжӢ·иҙқеҲ°жҢӮиҪҪзҡ„зӣ®еҪ•йҮҢ

жңҖеҗҺпјҢж·»еҠ зҺҜеўғеҸҳйҮҸдҪҝз”ЁиҮӘе®ҡд№үж•°жҚ®еә“и·Ҝеҫ„пјҢйҮҚж–°еҗҜеҠЁ

зҗҶжғіжғ…еҶөдёӢпјҢjoplin-serverдјҡдҪҝз”ЁжҢӮиҪҪзҡ„иҝҷдёӘж•°жҚ®еә“дҪңдёәеӯҳеӮЁпјҢз„¶иҖҢеҗҜеҠЁеҗҺжҠҘй”ҷпјҢзңӢй”ҷиҜҜдҝЎжҒҜжҳҜеӣ дёәжІЎжңүеҶҷжқғйҷҗпјҢжҲ‘еҸҲе°қиҜ•дҝ®ж”№ж–Үд»¶жқғйҷҗпјҢжё…зҗҶknex_migrations_lockиЎЁзҡ„ж•°жҚ®пјҢдҪҶжҳҜиҝҳжҳҜдёҚиЎҢгҖӮжңҖз»Ҳж”ҫејғдәҶжҢӮиҪҪиҝҷз§Қж–№ејҸгҖӮ

жҺҘзқҖеҸҲиҖғиҷ‘е…¶д»–ж–№жі•пјҢеҗҜеҠЁеҗҺдҪҝз”Ё/back/db-prod.sqliteиҰҶзӣ–й»ҳи®ӨеҲқе§ӢеҢ–зҡ„/home/joplin/packages/server/db-prod.sqliteпјҢдёҖиҜ•иҝҳзңҹиЎҢпјҢиҰҶзӣ–д№ӢеҗҺйҖҡиҝҮжөҸи§ҲеҷЁи®ҝй—®пјҢзҷ»еҪ•иҙҰеҸ·зЎ®е®һжҳҜдҝ®ж”№д№ӢеҗҺзҡ„пјҢз¬”и®°ж•°жҚ®д№ҹйғҪеңЁгҖӮйӮЈиҝҷз§Қж–№жі•жҳҜеҸҜд»Ҙзҡ„пјҢжҺҘдёӢжқҘе°ұеҘҪеҠһдәҶпјҢжҲ‘д»¬е®ҡжңҹеӨҮд»Ҫdb-prod.sqliteе°ұиЎҢпјҢеҸҜд»ҘеҶҷдёӘи„ҡжң¬еҗҜдёӘе®ҡж—¶д»»еҠЎпјҢе®ҡжңҹжҠҠж•°жҚ®еә“ж–Үд»¶еӨҮд»ҪеҲ°жҢӮиҪҪзҡ„дё»жңәзӣ®еҪ•дёӢгҖӮ
еҪ“з„¶е®ўжҲ·з«Ҝжң¬ең°ж•°жҚ®йғҪжңүзҡ„жғ…еҶөдёӢпјҢserverз«Ҝdockerе®№еҷЁз”ұдәҺжҹҗз§ҚеҺҹеӣ й”ҖжҜҒпјҢйҮҚж–°йғЁзҪІеҗҺпјҢд№ҹеҸҜд»ҘйҖҡиҝҮжң¬ең°иҰҶзӣ–иҝңзЁӢзҡ„ж–№ејҸејәиЎҢиҰҶзӣ–иҝңзЁӢж•°жҚ®гҖӮдёҚиҝҮиҝҷж ·жҖ»еҪ’жҳҜжңүдәӣйә»зғҰзҡ„пјҢжҜ•з«ҹйҮҚж–°йғЁзҪІserverз«ҜеҗҺиҝҳеҫ—дҝ®ж”№иҙҰеҸ·ж•°жҚ®гҖӮ

PostgresSQL еӨҮд»Ҫ

дҪҝз”ЁPostgresSQLзҡ„иҜқе®һйҷ…жҳҜеӨҮд»ҪPostgresSQLж•°жҚ®еә“пјҢdockerйғЁзҪІPostgresSQLзҡ„ж—¶еҖҷжҢӮиҪҪеҘҪи·Ҝеҫ„е°ұиЎҢпјҢиҝҷж ·dockerй•ңеғҸејӮеёёй”ҖжҜҒпјҢж•°жҚ®д№ҹдёҚдјҡеҸ—еҲ°еҪұе“ҚгҖӮ

жҖ»з»“дёӢжқҘпјҢиҝҳжҳҜPostgresSQLжӣҙж–№дҫҝдәӣпјҢж•°жҚ®е’Ңй•ңеғҸеҲҶеүІејҖжқҘпјҢжӣҙеҠ е®үе…ЁпјҢжһңж–ӯеҲҮеҲ°PostgresSQLдёҠжқҘгҖӮ
