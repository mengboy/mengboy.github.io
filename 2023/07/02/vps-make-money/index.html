闲置vps挂机赚钱回血

分享一下几个使用vps挂机赚钱的方案，楼主把基于docker就可以实现的都实操了下，其他一些用不了docker的简单列下仅供参考。只实操支持docker的主要是因为简单方便，注册下账号，机器上一个docker指令令跑起就可以了。这种挂机只能稍微回点血，想要盈利的可以绕开了。

流量挂机

  • traffmonetizer
    这是一款欧洲爱沙尼亚的流量挂机项目，10刀提现（Paypal、BTC、Payoneer），注册送5刀，支持Windows、Mac OS、安卓、docker，国内ip基本无收益。以下是安装指令，把your_token_here替换成下面截图中Your application token下的那串字符串。其他操作系统在截图中Download apps里下载相应app就行。

  • proxyrack

另外一款流量挂机的项目，支持windows和docker，实际上支持docker就可以在Windows、Linux、Mac OS下跑了。注册后在机器上执行以下指令

执行成功后，最后一条指令会输出一个uuid，在proxyrack网站里选Devices，把uuid填到Device ID一栏，Friendly name可以随便填，然后点击Add Device按钮，这里可能会有个延迟，docker跑起来后一两分钟后proxyrack才能识别到，识别到后才能添加成功。

广告联盟

  • 注册广告联盟oneptp
  • 获取广告链接，这里可以获取多个链接
  • 上面链接点击就有收益，一个IP最多点一次，多点不算，满0.01元即可提现到支付宝，微信、PayPal。那么该如何获取点击呢。下面介绍几个流量交换工具，通过这些工具去点，这里也只重点介绍支持docker部署的。

流量交换工具

  • otohits
    • 注册账号
    • 进入Applications界面，点击docker，下面会出现docker指令，执行跑起来
    • docker 跑起来后可以在setting界面添加要看的广告类型，可以都勾选上
    • 在My site页面添加上面的广告链接，普通用户最多添加3个，想添加更多就要买了
    • 添加完广告链接后，给每个链接分配点数，这些点数是一开始docker跑起来后otohits分配给你的广告播放完后获得的，otohits在发布你的广告链接时会消耗点数，这就是所谓的流量交换
    • 一般配好三个链接后，点数是用不完的，剩下的点数可以换钱
  • feelingsurf
    • 注册账号
    • 在earn credit界面拿到access token
    • 在服务上启动docker，XXXX换成你的access_token，运行成功后在earn credit界面可以看到运行情况
    • 在Website界面添加广告链接

      feelingsurf不需要配置点数，会自动交换，获取的点数otohits没有多，目前看也仅够配置一个广告链接的。
